Anything in machine learning and deep learning that you decide their values or choose their configuration before training begins and whose values or configuration will remain the same when training ends is a hyperparameter.

Here samples of hyperparameter:

  • Network Configuration

    • It consists of 3 layers, namely input layer, hidden layer, and output layer. The input layer contains input from the dataset, the hidden layer is in charge of training data, and the output layer shows the results of the training.

  • Neurons

    • Neurons in deep learning models are nodes through which data and computations flow. It flows from input layer to output layer.

  • Learning Rate

    • The learning rate is a hyperparameter that controls how much to change the model in response to the estimated error each time the model weights are updated. Choosing the learning rate is challenging as a value too small may result in a long training process that could get stuck, whereas a value too large may result in learning a sub-optimal set of weights too fast or an unstable training process.

  • Dropout Rate

    • Dropout is a technique where randomly selected neurons are ignored during training. They are “dropped out” randomly. This means that their contribution to the activation of downstream neurons is temporally removed on the forward pass, and any weight updates are not applied to the neuron on the backward pass.
